# Matcha Festival Day Operations Software

This is the day operations dashboard for the Matcha Festival, built with Next.js, Tailwind CSS, and Shadcn UI components.

## Features

- **Dashboard**: Real-time stats on revenue, tickets, and vendor status.
- **Vendor Management**: Track vendor approval, booth assignment, and status.
- **Schedule**: View the festival timeline.
- **Check-in**: Scan vendor badges or enter IDs for check-in.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS v4
- **Components**: Custom implementation of Shadcn UI patterns
- **Charts**: Recharts
- **Icons**: Lucide React
